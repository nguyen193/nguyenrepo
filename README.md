# How to run
---

## Cách chạy
```
git clone https://github.com/Vinhdtgcs200644/CloudProject.git
cd CloudProject
npm install
npm start
```

## Cách commit
```
git clone https://github.com/Vinhdtgcs200644/CloudProject.git
cd CloudProject
<chỉnh sửa v.v>
git add .
git commit <message>
git push
```
